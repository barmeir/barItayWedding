# Memory Images

Drop one image per stage here, named `stage-01.jpg` through `stage-12.jpg`.

If a file is missing, the app shows a styled gradient fallback so the experience
still works pre-asset-drop. Update the `image` paths in `src/data/stages.js` if
you want to use different filenames.

Recommended:
- Aspect ratio: 4:5 (portrait)
- Min width: ~720px
- Format: JPG (or PNG)
